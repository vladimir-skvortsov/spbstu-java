Instructions for assembling the application:
1. Install Java Development Kit (JDK) 8 or newer:
  https://www.oracle.com/cis/java/technologies/downloads/
2. Clone the repository:
  $ git clone https://github.com/modelware-ru/java-language/blob/main/Требования%20к%20практической%20работе.md
3. Go to the source code folder:
  $ cd ./haffman/src
4. Compile the application:
  $ javac Main.java
